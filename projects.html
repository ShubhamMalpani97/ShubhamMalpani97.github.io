End-to-End Credit Risk Modelling (PD, LGD, EAD)

Dataset: Lending Club 2007–2014 | Logistic Regression Scorecard | Regulatory Framework

Built a full Expected Loss framework estimating Probability of Default (PD), Loss Given Default (LGD), and Exposure at Default (EAD). Implemented Fine & Coarse Classing, Weight of Evidence (WoE), and Information Value (IV) for feature selection and model interpretability.

Model Performance (Out-of-Sample):
AUROC: 0.702
Gini Coefficient: 0.404
KS Statistic: 0.297

Developed a scorecard scaled between 300–850 for PD interpretation. Validation performed using out-of-sample testing with scikit-learn train-test split methodology. Model diagnostics included Gini coefficient, KS statistic, and CAP curve analysis to assess discriminatory strength and robustness.
